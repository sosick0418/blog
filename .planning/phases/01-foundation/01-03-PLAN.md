       - "build": "eleventy" — production build
       - "clean": "rm -rf _site" — clean build output

    11ty has browsersync built-in when using --serve flag, so no additional dependencies needed.

    Do NOT use npx in scripts — 11ty is installed locally, npm will find it in node_modules/.bin/

       - Open: false (don't auto-open browser)
